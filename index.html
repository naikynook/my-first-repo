Digital Object 1: Spatial Canvases

I chose to use the P5.js and three.js libraries to explore how code and web design can be used, perhaps with a projector, to create visual environments. Music visuals at concerts are a big inspiration and aspiration for what I hope code and web design can do for me.

Orbiting Grid Sphere

A stationary Three.js sphere wrapped in a white cell grid. Drag to orbit the camera around it, or scroll to zoom in and out. The grid texture is drawn on a canvas and mapped onto the sphere surface, while OrbitControls move the camera instead of rotating the object itself.

Interactive Sphere Matrix

An interactive Three.js sphere that behaves like a matrix panel wrapped in 3D. Use the sliders to change grid density and background color, scroll to zoom, and click the sphere to send a rainbow wave across individual cells. The code redraws a canvas texture each frame, measuring distance from the clicked cell to color only the squares inside a moving wave band.

Static Grid Configuration

A fixed 10×10 matrix of white squares on a black field with even spacing between cells. The canvas size is calculated from the number of columns, rows, square size, and spacing, then nested loops place each cell at its computed x and y position. The sketch runs once with noLoop(), treating the grid as a precise spatial layout rather than an animation.

2D Matrix Wave

A flat p5.js grid where each square acts as an individual cell in a matrix panel. Click anywhere to trigger a radiating rainbow wave from that point. On each frame, the code expands a circular wave radius and tests every cell’s distance from the click; only cells inside the wave front receive shifting HSB colors before returning to white.
